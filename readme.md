## Getting Started

### Prerequisites

Make sure you have the Rust toolchain installed on your computer and an understanding of basic concepts.

**Clone the project:**

```bash
git clone https://github.com/DevWajahat/Rust-1st-project.git
```

**Run The Project:**
```bash
cd Rust-1st-project
rustc main.rs
./main
```
